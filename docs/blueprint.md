# **App Name**: Base64 Forge

## Core Features:

- Secure Image Upload: A prominent, responsive drag-and-drop zone that accepts PNG, JPG, WEBP, and SVG formats, designed for client-side processing to ensure user privacy and data security.
- Instant Base64 Conversion: Client-side processing of uploaded images into Base64 strings using the FileReader API, ensuring fast conversion without server interaction as soon as the image is dropped or selected.
- Multi-Format Output: Provide dynamic output tabs or sections presenting the generated Base64 string in various useful code formats: raw Base64 string, HTML <img> tag, CSS background-image property, and the full Data URI.
- Code Interaction Utilities: One-click 'Copy to Clipboard' functionality for each output format and a 'Download as Text File' button to easily save the generated code snippets.
- Real-time Image Preview: Display a thumbnail preview of the uploaded image immediately upon selection or drag-and-drop, allowing users to verify the correct image before conversion.
- Thematic Mode Switch: A prominent UI toggle allowing users to switch between a clean dark mode and a light mode, enhancing accessibility and user preference for extended use.
- SEO Content Framework: Integrate structured placeholders for an H1 title, a concise descriptive introduction, and a dedicated FAQ section to support search engine optimization and organic user discovery.

## Style Guidelines:

- Primary color: A sophisticated, slightly desaturated violet-blue (#BAB5EE) that contrasts effectively against dark backgrounds, signifying a modern and digital aesthetic.
- Background color: A deep, muted aubergine-like gray (#1C1922) to establish a clean and professional dark mode interface, promoting focus on the tool's content.
- Accent color: A vibrant, clear blue (#7095F5) derived analogously from the primary hue, used sparingly for interactive elements, calls-to-action, and highlights to provide clear user feedback.
- Body and headline font: 'Inter' (sans-serif) for its modern, objective, and neutral appearance, ensuring excellent readability and a 'SaaS-style' feel across all text.
- Utilize a consistent set of minimal, functional line icons for actions like upload, copy, and download, matching the clean SaaS-style UI. Prioritize clarity and simplicity in all icon designs.
- A single-page, responsive layout with a prominent, visually engaging drag-and-drop upload area. Output formats should be organized using a clear tab-based system for easy navigation, alongside a dynamic image preview.
- Incorporate subtle, tasteful animations for user feedback, such as hover effects on interactive elements, visual cues during drag-and-drop, and a discreet confirmation for 'Copy to Clipboard' actions.
